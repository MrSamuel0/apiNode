# Node API

A RESTful API built with Node.js, TypeScript, and Express.

This project follows a layered architecture to keep responsibilities separated and the codebase scalable.

---

## 🚀 Tech Stack

- Node.js
- TypeScript
- Express

---

## 📁 Project Structure

```bash
controllers/   # Request handlers (HTTP layer)
dto/           # Data Transfer Objects (input/output contracts)
models/        # Data models / interfaces / schemas
routes/        # API route definitions
services/      # Business logic layer
server.ts      # Application entry point (Express setup)
```
